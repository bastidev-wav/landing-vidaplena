## Descripción
Landing Page desarrollada para VidaPlena.

## Tecnologías utilizadas
- HTML5
- CSS3
- Git
- GitHub

## Instrucciones de uso
Abrir el archivo index.html en un navegador web.

## Prompt utilizado

Genera 3 testimonios ficticios para una empresa llamada VidaPlena SpA, dedicada a servicios de bienestar y salud integral. Cada testimonio debe incluir nombre completo y un párrafo breve con una experiencia positiva relacionada con bienestar físico, salud mental o hábitos saludables.

## Uso de IA

Se utilizó Github Copilot para:
- Generar ideas de contenido para la sección de testimonios.
